
조건문
if/ else if /else 등의 문장을 사용해서 조건에 따라 다른 코드 블록 실행

주어진 조건을 평가하고 참일 경우 특정 코드 실행. 거짓일 경우 다른 코드 실행.

1) if

	if (조건문) {
		참(true)이면 실행
		} 거짓(false)면 종료


2) if-else

	if (조건문) {
		//조건이 참일 때 실행되는 코드
		} else {
		//조건이 거짓일 때 실행되는 코드
		}


3) if / else if / else

여러개의 조건을 순차적으로 평가하고, 그 중 하나의 조건에 해당하는 코드블록을 실행하는 구조
모든 조건을 차례대로 검사하며, 첫 번째로 참이 되는 조건에 해당하는 코드블록만 실행되고 나머지 조건은 무시

여러개의 if 조건을 연속적으로 검사하면서, 첫 번째로 참이 되는 조건의 코드블록을 실행
else if는 이전 조건이 거짓일 대만 실행되며, 여러 개의 조건을 추가할 수 있다~

마지막 else 블록은 이전 모든 조건이 거짓일 때 실행

	if (조건1) {
		//조건1이 참일 때 실행되는 코드
		} else if (조건2) {
		//조건2가 참일 때 실행되는 코드
		} else {
		//이전 모든 조건이 거짓일 때 실행되는 코드
		}
