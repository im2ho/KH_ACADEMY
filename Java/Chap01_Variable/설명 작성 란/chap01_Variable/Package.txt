package란 어떠한 폴더 위치를 작성하는 공간
내가 만들고싶은 파일 경로(폴더 위치)를 설정할 수 있음

new > class(아마도.. 작업공간..) > package 수정 가능
ex) com.kh.variable: 폴더 3개가 자동 생성(com/kh/variable)


*****자바의 명명 규칙*****
자바의 명칭 규칙은 클래스나 패키지 변수 상수 메소드 등.. 자바 프로그래밍을 할 때 이름을 따라할 규칙

1) 카멜케이스(javaName): 이름에 여러 단어를 혼합해서 사용한다면 첫 번째 단어는 소문자, 두 번째 단어는 대문자로 시작, 이름을 지을 때 숫자를 먼저 입력해도 안됨.. (ex: 1004nany)
 
 ex) firstName / lastName / phoneNumber

2) 스네이크케이스: 모든 문자의 첫 자는 소문자로 표기하고, 문자들 사이들은 _(언더바)로 연결
 
 ex) frist_name / last_name / phone_number
 
3) 파스칼 케이스: 모든 문자의 첫 자를 대문자로 표기, 중간에 대문자를 넣어 표기 가능
 
 ex) FirstName / LastName / PhoneNumber
 
4) 케밥케이스: 모든 문자의 첫 자는 소문자로 표기하고 문자들 사이에는 -(하이픈) 연결
 
 ex) fisrt-number / last-number / phone-number
